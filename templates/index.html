Methodology and honest limitations

        
    
            
  • Engagement data is a hand-seeded dataset ({{ stats.engagement_records }} rows) used to validate the aggregation + clustering pipeline.
    • 
            
  • BERT sentiment uses distilbert-sst-2 — pre-trained on movie reviews; not fine-tuned on fashion captions.
    • 
            
  • CNN trained on public product photos; deployed accuracy on real social-media imagery is expected to be lower without domain fine-tuning.
    • 
            
  • Trend "forecast" endpoint uses linear slope detection; the LSTM training code exists for Phase 2 once time-series data is collected.
    • 
            
  • Phase 2 scope: Lagos-collected social media data, alté category, real engagement time series, fine-tuned CNN.
